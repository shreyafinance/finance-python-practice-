# Finance + Python Practice

This repository contains beginner Python scripts applied to finance, compliance, and FreeCodeCamp projects.

## Contents
- **finance_basics/** → Scripts related to MIS automation, compliance workflows, and regulatory data.
- **workshop_projects/** → Practice projects like Ticket Calculator, Bill Splitter, Travel Weather Planner, and Caesar Cipher.
- **freecodecamp_exercises/** → Python practice exercises from FreeCodeCamp (variables, loops, functions, conditionals, string operations).
- **certification_projects/** → Major FreeCodeCamp certification projects:
  - Build a User Configuration Manager
  - Build a Budget App
  - Build a Polygon Area Calculator
  - Build a Hash Table
  - Implement the Tower of Hanoi Algorithm
